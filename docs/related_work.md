# Related Work & Différenciation

## Frameworks proches

### AutoGen
Permet la collaboration entre agents via conversations structurées.

### CrewAI
Met l'accent sur les rôles et les workflows métier.

### LangGraph
Permet de construire des graphes agentiques avec cycles et reprise.

## Différenciation de Limen

Limen ne cherche pas seulement à faire collaborer plusieurs agents.
Il cherche à déterminer **quand la collaboration est nécessaire**.

Autrement dit :

- les frameworks classiques optimisent la collaboration ;
- Limen optimise la décision de collaborer.

## Angle spécifique

L'innovation principale du projet se situe dans l'articulation de trois idées :

1. estimation de compétence ;
2. politique de seuil explicable ;
3. mémoire commune de projet.
